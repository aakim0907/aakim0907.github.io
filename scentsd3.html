Project Description

Scents of Serge Lutens is a data visualization project showing the prevalence and links between fragrance notes of Serge Lutens' main-line perfumes. This application was built using JavaScript (node.js, d3.js) and HTML/CSS.

Information about fragrance notes were scraped using node.js and was rendered using d3.js. Then, Fragrance notes were represented as data nodes. Notes that appear in same perfume are connected, and an event handler was implemented for dynamic and interactive visualization.
