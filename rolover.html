Project Description

Rolover is a full-stack web application inspired by Rover. Rolover enables users to search for dog trainers according to different training types and send a request to book a training session with a trainer. This web application was designed and built in 3 weeks.

Feature highlights:

  1. Secure Authentication
  2. Trainer Search
  3. Send a booking request to a trainer
  4. Leave a review to a trainer
  5. User profile page
